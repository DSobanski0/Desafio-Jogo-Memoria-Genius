# Desafio-Jogo-Memoria-Genius

Desafio da Digital Innovation One utilizando HTML, CSS e JavaScript

Nesse projeto, o desafio foi criar um jogo da memória estilo Genius utilizando apenas HTML, CSS e Javascript. Os principais conceitos utilizados foram CSS Grid, manipulação de Arrays e Arrow Functions.

Realizei uma mudança em relação ao projeto inicial em que a cor que ficava mais clara não voltava ao seu estado anterior, impossibilitando a visualização da sequencia a ser clicada. Divirta-se!
